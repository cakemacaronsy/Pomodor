# 鳳梨時鐘 Medium v1

TL;DR：這是中等強度版鳳梨時鐘的獨立版本。它保留專注、短休息、長休息、今日完成數、任務清單與本機儲存。

## 版本內容

| 項目 | 說明 |
| --- | --- |
| 版本 | `pomodoro-medium_v1` |
| 入口 | `index.html` |
| 樣式 | `styles.css` |
| 功能 | `app.js` |
| 圖示 | `assets/pineapple.svg` |

## 使用方式

1. 直接打開 `versions/pomodoro-medium_v1/index.html`。
2. 或在 `versions/pomodoro-medium_v1/` 執行：

```bash
python3 -m http.server 4173
```

然後開啟 `http://127.0.0.1:4173/`。
