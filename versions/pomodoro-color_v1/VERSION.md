# 番茄時鐘 Color v1

TL;DR：這是不同顏色版本的獨立番茄時鐘。它使用綠色視覺系統與葉子圖示，功能與基本番茄鐘一致。

## 版本內容

| 項目 | 說明 |
| --- | --- |
| 版本 | `pomodoro-color_v1` |
| 入口 | `index.html` |
| 樣式 | `green.css` |
| 功能 | `green.js` |
| 圖示 | `assets/green-leaf.svg` |

## 使用方式

1. 直接打開 `versions/pomodoro-color_v1/index.html`。
2. 或在 `versions/pomodoro-color_v1/` 執行：

```bash
python3 -m http.server 4173
```

然後開啟 `http://127.0.0.1:4173/`。
