# 綠意番茄鐘 v1

TL;DR：這是綠色版番茄時鐘的 v1 固定版本。它保留三段式番茄鐘、任務清單、今日完成數、本機儲存與響鈴提示，並使用綠色視覺系統與葉子圖示。

## 版本內容

| 項目 | 說明 |
| --- | --- |
| 版本 | `pomodoro-green_v1` |
| 入口 | `index.html` |
| 樣式 | `green.css` |
| 功能 | `green.js` |
| 圖示 | `assets/green-leaf.svg` |

## 使用方式

1. 直接打開 `versions/pomodoro-green_v1/index.html`。
2. 或在 `versions/pomodoro-green_v1/` 執行：

```bash
python3 -m http.server 4173
```

然後開啟 `http://127.0.0.1:4173/`。
